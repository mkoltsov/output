Raspberry Pi 400 shortlist

Games worth trying on this machine

A practical pick list for Nintendo 64, PSP, Dreamcast, and Sega Saturn emulation on Raspberry Pi 400-class hardware. It favors games that are both worth playing and relatively sensible targets for this performance level.

1. Nintendo 64Best fit among these platforms. Start here.
2. PSPGood for lighter games with PPSSPP-style emulation.
3. DreamcastWorth trying, but tune per game.
4. Sega SaturnMost experimental. Prefer 2D and slower-paced games.
N64

Nintendo 64

Strongest recommendation for this hardware, though some games still need emulator/plugin tuning.

Start first

  • Super Mario 64
  • Mario Kart 64
  • Star Fox 64
  • F-Zero X
  • Wave Race 64
  • The Legend of Zelda: Ocarina of Time
  • Super Smash Bros.
  • Diddy Kong Racing
  • Kirby 64: The Crystal Shards
  • Sin and Punishment
  • Mario Tennis
  • Mario Golf

Try later

  • Banjo-Kazooie
  • Donkey Kong 64
  • Paper Mario
  • GoldenEye 007
  • Perfect Dark
PSP

PlayStation Portable

A good second target. Start with 2D, tactics, rhythm, and racing games before heavier 3D titles.

Good targets

SAT

Sega Saturn

Lowest-confidence platform here. Treat this as an experiment, not a dependable library.

Start with these

  • Saturn Bomberman
  • Guardian Heroes
  • Street Fighter Alpha 2
  • Street Fighter Alpha 3
  • X-Men vs. Street Fighter
  • Marvel Super Heroes vs. Street Fighter
  • Sega Rally Championship
  • Virtua Fighter 2
  • Panzer Dragoon
  • Panzer Dragoon II Zwei
  • NiGHTS into Dreams
  • Dragon Force
  • Shining Force III
  • Albert Odyssey

Saturn emulation is unusually demanding for the hardware. If a game stutters or glitches badly, that result is expected rather than a bad dump by default.

Use dumps from games you own. Recommendations are based on Raspberry Pi 400-class limits and public emulator compatibility notes, not a guarantee that every title will run perfectly.
